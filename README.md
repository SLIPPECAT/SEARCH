# SEARCH
## 검색이란?
- 검색은 검색을 하는 대상이 원하는 내용을 찾아내는 행위이다.
- '좋은 검색'이란, 사용자의 의도에 부합하는 검색 결과를 내놓을 확률이 높은 검색이다.
- 이를 위한 것이 검색 엔진이며 검색 엔진의 성능 향상을 통해 사용자들의 사용 경험 향상 및 사용률 증가를 발생시킬 수 있다.

### 검색을 위한 세 가지 과정
1. 색인
2. 검색(또는 질의)
3. 스코어링(또는 랭킹)
